## Notes

- Single "Wahlen" FeatureServer hosts one current snapshot (layer 0 = Wahlbezirke, 91 polygons; layer 1 = Wahlbereiche; layer 2 = Wahlkreise). 91 WBZ matches expected BTW polling-district count for OL (~120 ceiling generous; OL had ~92 WBZ at BTW21) — confirms this is true WBZ, not the 9 Stadtbezirke or 34 Stadtteile.
- Polygons only — current boundaries; no time-series/archive of historical layouts published. Last service edit 2024-09-20 (pre-BTW25), so this is the BTW25 layout.
- For BTW 2013/2017/2021 historical WBZ geometries, must email gis@stadt-oldenburg.de (cc wahlbuero@stadt-oldenburg.de) and request archived shapefiles per election year.
- License DL-DE/BY-2.0: free reuse with attribution "© Stadt Oldenburg, Fachdienst Geo und Daten, dl-de/by-2-0".
- Attribute fields contain WBZ ID (BezirksNr) and polling-station ID (Wahllokal_ID) but no election-year attribute — confirms single snapshot.
- Also listed on opendata.oldenburg.de (https://opendata.oldenburg.de/dataset/wahlen) which mirrors the ArcGIS Hub.

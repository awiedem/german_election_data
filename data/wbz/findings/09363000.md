## Notes

- Weiden i.d.OPf. has 23 Urnen-Wahlbezirke (confirmed via BTW 2025 result tables); BTW 2013/2017/2021 likely the same order of magnitude (Bayer. LfStat does not centralize WBZ geometries — responsibility lies with the municipality).
- No general open-data portal for Stadt Weiden. Statistical office (bildung@weiden.de) publishes only PDF reports; no GIS/CSV downloads.
- BürgerGIS (Ingrada) currently exposes Bebauungspläne only — Wahlbezirk layer would have to be activated by Vermessungsabteilung if it exists internally.
- Recommended outreach: single email to josef.neumann@weiden.de copying wahlen@weiden.de, requesting WBZ polygon shapefiles (or GeoJSON) for BTW 2013, 2017, 2021, 2025 with licensing terms (DL-DE-BY-2.0 preferred). gutachterausschuss@weiden.de is a fallback if Herr Neumann is out.
- No prior FragDenStaat IFG request found against Stadt Weiden for WBZ geodata; Bayer. IFG (BayDSG / Transparenzgesetz) is a fallback if direct request is denied.
- ADBV Weiden (state surveying office) holds ALKIS/ALK and DOPs but not municipal WBZ — listed only as reference.

## Notes

- Two distinct WBZ layers are published: **Stimmbezirke** (172 polygons, used for BTW/LTW; this is the WBZ-equivalent) and **Kommunalwahlbezirke** (33 polygons, used only for municipal council elections). Use the Stimmbezirke layer for federal-election work — the count (~172) is in line with the expected ~200 WBZ for Münster.
- The "current" layer is hosted under the city's `61_stadtentwicklung` directory (Stadtentwicklung/Statistik), not on the open-data CKAN server itself; the CKAN dataset just links out. Updated 2024-11-11.
- For BTW 2013 specifically, the 2018-stand file is the closest published snapshot. If exact 2013 boundaries are required, contact opendata@citeq.de — the dataset description states older versions are available on request.
- 2025 BTW boundaries should be re-verified against any post-Nov-2024 update; the Kommunalwahl 2025 redistricting (decided 14 Jan 2025) affected Kommunalwahlbezirke (33), which is independent of the Stimmbezirke layer.
- Format is shapefile (.zip) for all years; no GeoJSON/GPKG/WFS variants observed on the portal.

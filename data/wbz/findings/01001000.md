## Notes

- Verified via direct `curl` GET: BTW2021 file is 50 KB, valid GeoJSON with 44 MultiPolygon features in WGS84; BTW2025 file is 79 KB, same schema, 44 features. The 22 Briefwahlbezirke are folded into the same election-results page as data rows but have NO polygon entries — Flensburg only publishes Urnen-WBZ geometries.
- The Flensburg city ArcGIS REST root (https://geoportal.flensburg.de/arcgis/rest/services) was fully enumerated — its sub-folders `__Geobasisdaten`, `_weitere`, `Utilities` contain MapServers for ALKIS, Luftbilder, Bauleitplanung, Bürgerbeteiligung, Statistik-Kleinräumige Gliederung, etc. — but **no Wahl/Wahlbezirk MapServer or FeatureServer is exposed**. So the only public source is the per-election GeoJSON embedded in the formular.flensburg.de results system.
- ckan.flensburg.de (the city's CKAN portal advertised on flensburg-mitmachen.de) was unreachable at probe time (DNS/HTTP both down) — likely pilot/abandoned; no Wahlbezirk dataset documented in OK Lab's open-data-api repo either.
- BTW 2013 + 2017 polygons are **not published anywhere we could find**. WBZ count (44 Urne + 22 Brief = 66) and polling-place names are stable across 2021↔2025, so the 2013/2017 layout is plausibly very similar — but municipal redistrictings between 2013 and 2017 cannot be ruled out without direct confirmation. Recommended ask to hillebrand.oliver@stadt.flensburg.de: "Wahlbezirksgeometrien BTW 2013 und BTW 2017 als Shapefile/GeoJSON; ggf. Hinweis auf Stadtvermessungsamt/Geodatenstelle".
- Dedicated `Stadtvermessung` / `Geodatenstelle` inbox could not be confirmed (the Daten- und Geoportal page is a 404; flensburg.de has no organisational unit named `Vermessung`/`Geoinformation` in its A-Z listing). The IT department is the de-facto contact per the geoportal page; route through `stadtverwaltung@flensburg.de` if Wahlamt cannot supply the geometries.
- Number of WBZ matches user's "~60 expected" exactly: 44 Urne + 22 Brief in 2021, 44 Urne + 24 Brief in 2025 (Brief count rose, polling-place layout unchanged).

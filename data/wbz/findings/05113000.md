## Notes

- Dataset publisher credits both FB 62 (Geo) and FB 12 (Statistik/Wahlen), so write to both for years not online.
- 309 Stimmbezirke matches the expected Urnenwahlbezirk granularity for Essen (~400 estimated in task description was an upper bound; actual BTW2021 count is 309).
- Open data request from Apr 2024 (Volt Deutschland / Arno Waschk) confirmed Briefwahl-geometries are not published; only Urnen-Stimmbezirke are. Status: "Datensatz steht bereit" but not for BTW2025/historic years.
- For 2013/2017 historic geometries, FragDenStaat IFG-Anfragen to Wahlamt may be the only path; mention dl-de/by-2.0 as preferred license.
- Kataster (kataster@amt62.essen.de) handles cadastre, not election geometries — use geoinformation@amt62.essen.de for WBZ.

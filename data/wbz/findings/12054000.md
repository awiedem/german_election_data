## Notes

- Strong coverage: BTW 2021 + BTW 2025 are both on the open-data portal as shapefile/GeoJSON, multi-format. Catalog API confirms the dataset IDs.
- Gap: BTW 2013 and BTW 2017 not on portal. Portal also has no Landtag/Kommunalwahl geodata older than 2019, suggesting Potsdam started publishing WBZ geometries ~2018 (earliest dataset: OB-Wahl 2018). Older boundaries likely exist as internal GIS layers in the Bereich Statistik und Wahlen.
- Recommended request route for 2013/2017: email Statistik@Rathaus.Potsdam.de (primary, owns WBZ definitions) cc Geodatenservice@Rathaus.Potsdam.de.
- Feature count not explicitly verified by downloading; ~130 inferred from Potsdam OB-Wahl 2025 (131 districts: 90 Urne + 41 Brief).
- Adjacent useful dataset: `strassen_wahlbezirke` (street-to-WBZ assignment) on same portal — could in principle be used to reconstruct older geometries via address geocoding if shapefiles unavailable.
- All datasets accessible via OpenDataSoft REST API at `https://opendata.potsdam.de/api/explore/v2.1/catalog/datasets/<dataset-id>/exports/<format>`.

## Notes

- Bayreuth (city, ~74k inhabitants, kreisfrei) has NO dedicated municipal open-data portal. The only "open data" page found is `o-sp.de/bayreuth/opendata.php` which lists exactly one dataset (Bebauungsplanübersicht via WMS/WFS at `geodata.o-sp.de/service/bayreuth.cgi`). No election layer.
- The City has NO own Stadtvermessungsamt or Geoinformation department — confirmed via the A–Z department directory. Cadastral work is done by the state ADBV Bayreuth (poststelle@va-bt.bayern.de, Wölfelstr. 2), but ADBV handles Liegenschaftskataster only and is NOT relevant for municipal election geometry (same conclusion as Ingolstadt finding).
- Election infrastructure runs on AKDB ok-vote (`online-dienste.bayreuth.de/wahlen/...`). The 2019 EW page (`OpenDataInfo.html`) confirms only tabular CSV exports are offered (`opendata-wahllokale.csv`, `Open-Data-Europawahl133.csv`, `Open-Data-Europawahl136.csv`) — explicitly NO street directory and NO polygons. No equivalent OpenData page found for BTW 2017/2021/2025; only HTML result trees.
- Stimmbezirk IDs visible in 2025 result URLs (e.g. `094620000019`) confirm internal numbering 0001–~0050 in-person + Briefwahl bezirke. Bayreuth has had ~46–52 Urnenwahlbezirke in recent BTW (consistent with ~50 expected). The polygons exist internally but are not published.
- Bayern open-data portal `open.bydata.de` aggregates city portals; search for Bayreuth WBZ returns nothing. State Geoportal Bayern (`geoportal.bayern.de`, `geodaten.bayern.de/opengeodata/`) provides Verwaltungsgebiete (Gemeinde level) and ALKIS but no municipal Wahlbezirke.
- For Bayern as a state, Stimmkreise (Landtagswahl) were obtained via FragDenStaat (`fragdenstaat.de/.../geometrien-der-stimmkreiseinteilung-zur-landtagswahl-2018-in-bayern-...`) — same route may work for BTW WBZ at Bayreuth city level, but it would be an IFG request to the City rather than the state.
- Recommended outreach order: (1) stadtplanungsamt@stadt.bayreuth.de — most likely holder of internal GIS layers, since the city contracts the WMS via o-sp.de via Stadtplanungsamt; (2) wahlamt@stadt.bayreuth.de cc 0921 25-1208 (deputy head, Wahlen) — holds the Wahlbezirkseinteilung as administrative boundaries; (3) grundstuecksamt@stadt.bayreuth.de — fallback if first two redirect. Request all four cycles (BTW 2013/17/21/25) in one message; ESRI Shapefile or GeoJSON acceptable.

## Notes

- WBZ shapefile verified via download: ZIP 99,262 bytes; contains `Wahlbezirke21_komplett.{shp,shx,dbf,prj,cpg}`; ogrinfo reports Feature Count 99, Polygon, EPSG:25832 (ETRS89/UTM32N). 99 features is consistent with Urnenwahlbezirke for a city of ~160k inhabitants and clearly NOT the 9 Stadtteile or 37 statistische Stadtteile.
- Same ZIP is referenced from both the BTW2025 dataset and the Kommunalwahlen 2026 dataset; appears to be the city's currently-published WBZ geometry. No separate historical geometries (BTW 2013 / 2017 / 2021) are published on opendata.darmstadt.de — the 2021 election has only an interactive HTML viewer plus CSV/JSON time series. Pre-2021 WBZ geometries are not online and would need to be requested directly.
- Dataset page warns the shapefile is only provisional until the Darmstadt Geoportal launches; URL may change. Geoportal Hessen (geoportal.hessen.de) does not currently host municipality-level Darmstadt WBZ.
- For BTW2013 and BTW2017 boundaries, recommended request path: vermessungsamt@darmstadt.de (geo primary) cc wahlen@darmstadt.de; offenedaten@darmstadt.de can broker if Vermessungsamt is unresponsive.
- BTW2021 shapefile is presumed identical to the published BTW2025 geometry; cross-check WBZ codes against the 2021 results CSV to confirm before using as 2021 boundary.

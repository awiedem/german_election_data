## Notes

- BTW 2017 had 95 WBZ, BTW 2013 had 94 WBZ (per heidelberg.de result pages) — feature counts above are based on these official figures, not direct geometry inspection.
- Heidelberg posts a per-election Shape-Datei link on each BTW subpage; the 2013 page predates this practice. Recovering 2013 boundaries likely requires an FOI/email request to Wahldienststelle or Vermessungsamt.
- 2025 file is published as GeoJSON only (path includes "GRW_2024" — same boundaries reused from the 2024 Gemeinderatswahl); a sample WebFetch returned only 6 features from the file (likely a truncation/parse artifact since the listed size is 648 KB) — re-download and verify locally before using.
- daten-bw.de listing for BTW 2025 only exposes a CSV of results (license "Eingeschränkte Nutzung"), not the geometry; geometry lives on heidelberg.de.
- Geoportal: https://geoweb.heidelberg.de/geoportal — contains a "Wahlbezirke" layer for visualization; no public WFS endpoint surfaced.
- Abteilung Statistik (statistik@heidelberg.de) is the most likely owner of historical boundary files; GIS@heidelberg.de for technical/format questions.

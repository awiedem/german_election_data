## Notes

- Single combined dataset "Wahlen" covering Stimmbezirke (polygons) and Wahllokale (points). 180 features each — slightly below the user-quoted ~190, plausibly because some Stimmbezirke share polygons (e.g., split by mail-in/Briefwahl handled separately) or count drifted between elections. Still WBZ-level (not the 4 Stadtbezirke or 44 Stadtteile).
- Geoportal hotlink path `/wms/Hotlinks/OpenData/` is not directory-listable (HTTP 403) — only the published filename is reachable. No archived per-year snapshots are visible.
- BTW 2013 / 2017 geometries: not on open portals. FB 62 (gdi@) is the right inbox; the Wahlamt likely holds historical Wahlbezirk lists but probably as PDF/CSV without geometry. ITK Rheinland (https://wep.itk-rheinland.de/) hosts MG election results but not geodata.
- CKAN harvester record shows last metadata refresh 2026-02-06 but original `modified` is 2022-06-14 — i.e., the data file itself has not been re-versioned since the Kommunalwahl-2020 / BTW-2021 cycle. Worth asking FB 62 whether a 2025 snapshot exists internally.
- License is fully open (Zero 2.0); no attribution required for reuse but courtesy attribution to "Stadt Mönchengladbach, FB 62" is standard.

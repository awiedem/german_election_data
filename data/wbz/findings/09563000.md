## Notes

- Fürth is split into 92 Stimm-/Wahlbezirke per the Statistikamt page (https://www.nuernberg.de/internet/statistik/fuerth_stimm_wahlbezirke.html). The published ArcGIS layer has 98 features — likely 92 Urne plus 6 Brief/aux polygons. NOT confused with the 13 statistische Bezirke or the smaller set of Stadtbezirke (separate datasets in the same portal).
- Statistics for both Nürnberg and Fürth are produced by the joint "Amt für Stadtforschung und Statistik Nürnberg und Fürth"; this is the actual data steward for the shapefile, even though Fürth's own portal hosts the download.
- For BTW 2013/2017/2021, the only documented path is an explicit request. Two parallel routes worth trying: (1) Statistikamt's "statistical data" contact form (custom extracts ≥ €60, with academic exemption); (2) Stadt Fürth Vermessung (spa.vm@fuerth.de) plus Wahlamt (wahlen@fuerth.de) — Vermessung holds the geometry tooling; Wahlamt knows which historical WBZ-Schnitt was in force per election cycle.
- Boundary stability: the 2025 layer is "Stand 06.03.2024" and applies to BTW 2025 only. Re-zonings happen between cycles, so 2025 polygons should not be used as a proxy for 2013/2017/2021.
- govdata.de search yields zero datasets under the "stadt-fuerth" organization and zero matches for "Wahlbezirk Fürth"; bayernportal/geodaten.bayern.de carry no city-level WBZ.
- ArcGIS Online global search returns only the single 2025 item id `a32552409fb047e2bc758bcee0b36a8b`; no older Fürth WBZ items have been published.
- Sister city note: Nürnberg (09564000) shares the same Statistikamt and the same request-only policy for pre-2025 vintages.

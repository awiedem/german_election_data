## Notes

- Single-resource portal: only one current `wahlbezirke.gpkg` is exposed; historical BTW 2013/2017 versions are not published. Past versions may exist internally at Statistisches Amt or Stadtmessungsamt — request via poststelle.12-opendata@stuttgart.de.
- The dataset is hosted by Statistisches Amt (publisher); geometry production is done by Stadtmessungsamt. For raw geometry / archive snapshots, Stadtmessungsamt is the better contact.
- Feature count not displayed on portal; must download GPKG to verify ~400 polygons (Stuttgart BTW 2021 had ~373 Urnen-WBZ).
- The dataset mixes Urnen- and Briefwahlbezirk identifiers as attributes on a single geometry layer — the polygons themselves are Urnenwahlbezirke (Briefwahl is non-spatial reference).
- "Last updated April 24, 2026" on portal page postdates BTW 2025 (Feb 2025) → file likely now holds 2025 boundaries; the 2021 version is presumably overwritten. To recover BTW 2021 specifically, ask Stadtmessungsamt for the archived 26.09.2021 snapshot.
- Geoportal Stuttgart (https://www.stuttgart.de/leben/bauen/geoportal/) returned 403 to direct fetch but is the formal landing page; also lists Open-Data Geodaten via the Stadtmessungsamt.
- BW state geoportal (geoportal-bw.de) and govdata.de were not separately checked; Stuttgart routes its WBZ data through its own portal, so duplicates upstream are unlikely to add coverage.

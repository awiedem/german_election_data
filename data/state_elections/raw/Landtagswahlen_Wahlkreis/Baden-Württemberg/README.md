# Baden-Württemberg — Landtagswahl results at constituency (Wahlkreis) level

**State:** Baden-Württemberg (state_abbr: BW)
**Constituency unit:** Wahlkreis (70 Wahlkreise since 1976; fewer in earlier elections — the
state was newly formed in 1952, the modern unified-Wahlkreis Landtag election series begins 1956).
**Vote system note:** From 1956 through 2021 the BW Landtagswahl used a **single vote** (one
Erststimme per voter; list/Zweitmandate ("Zweitauszählung") derived from the same vote). Starting
with the **2026** election BW introduced a **two-vote system** (Erststimme + Zweitstimme); the 2026
open-data file therefore carries both Erststimmen (D-columns) and Zweitstimmen (F-columns).

This folder holds RAW source files, stored verbatim, for the GERDA research database.

## Primary sources

1. **Statistische Bibliothek** (digital archive of the official statistical offices),
   series *"Statistische Berichte / B / VII / 2 / (2) — Endgültige Ergebnisse der Wahl zum Landtag
   von Baden-Württemberg ... nach Wahlkreisen"*, series id `BWSerie_mods_00000906`.
   Browse: https://www.statistischebibliothek.de/mir/receive/BWSerie_mods_00000906
   This is the authoritative, complete back-catalogue (1956–2026). Each issue is the official
   constituency-level final-results report (PDF; 2016/2021/2026 additionally as XLSX).
2. **Statistisches Landesamt Baden-Württemberg** (statistik-bw.de) — current open-data files
   (2021 fileadmin CSV; 2026 election-presentation open data CSV).
3. **wahlen.statistik-bw.de/ltw26** — 2026 election-night/final open-data presentation portal.

## Format notes

- **2021 CSV** (`LTW21_Kreise.csv`): despite the "Kreise" filename, the first column is `Wahlkreis`
  and the 70 data rows are the constituencies ("01 Stuttgart I" … "70 …"). Columns: Wahlberechtigte,
  Wähler, ungültige/gültige Stimmen, then one column per party (GRÜNE, CDU, AfD, SPD, FDP, DIE LINKE,
  ÖDP, …). ISO-8859 / CRLF.
- **2026 CSV** (`ltw26-ergebnisse.csv`): the official all-levels open-data file. 12,741 rows covering
  Land → 70 Wahlkreise → 1,101 Gemeinden → ~11,570 Wahlbezirke. Field semantics are in
  `BW_2026_Hinweise-Datensatzbeschreibung.pdf`. Take rows where `Gebietsart == WAHLKREIS`.
- **XLSX (2016/2021/2026)**: the official report workbook; one worksheet per Wahlkreis plus
  state-summary sheets. Header row inside each sheet, "Wahlkreis 01    Stuttgart I" etc.
- **PDF reports (1956–2026)**: the official "B VII 2" final-results bulletins; tabular Wahlkreis
  results requiring OCR/table extraction. Best machine-readable substitutes: XLSX from 2016 onward,
  CSV for 2021 & 2026.

## Special / historical entries

- **1956** and one of the **1960** files are titled *"… umgerechnet auf die Wahlkreise der kommenden
  Bundestagswahl"* — i.e. the Landtag result recomputed onto the **federal (Bundestag) Wahlkreis**
  geometry, not the Landtag Wahlkreis geometry. They are still constituency-level Landtagswahl data
  and are kept for completeness. The 1960 `…_endgueltig.pdf` is the standard Landtags-Wahlkreis report.
- **1961** is a *Wiederholungswahl* (court-ordered repeat) in only Wahlkreise Waiblingen I & II — a
  partial by-election, not a full statewide Landtagswahl.

## Coverage summary

Complete BW Landtagswahl constituency-level coverage **1952 → 2026** (every election to the Landtag /
Verfassunggebende Landesversammlung of the unified state: 1952, 1956, 1960, 1964, 1968, 1972, 1976,
1980, 1984, 1988, 1992, 1996, 2001, 2006, 2011, 2016, 2021, 2026), plus the 1961 Waiblingen
Wiederholungswahl. Machine-readable (CSV/XLSX) for 2016, 2021, 2026; PDF for all earlier years.
**No gap remains** — 1952 is now obtained as a dedicated Wahlkreis report. The earliest unified-state
Baden-Württemberg election (1952, the constituent-assembly vote that created the state) is the
theoretical earliest, and it is now covered.
